# Life Insurance Contracting Platform

## Tecnologies

* C# 8
* .NET Core 3.1
* ASP.NET Core
* Swagger 5
* EntityFramework Core 3.1
* xUnit 2
* NSubstitute 4
* Coverlet
* FluentAssertations 5
* Angular 8
* Typescript 3.7

## Deployment

1. Build & Run Insurance.Api (http://localhost:5000)

2. Build & Run Insurance.Web (http://localhost:4200)
